# VacationAlly App Backend (Spring Boot + MySQL)

## Overview
Spring Boot backend for a vacation booking system supporting customers, vacations, excursions, carts, and checkout functionality through REST APIs.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven


## My Contributions
- Designed and implemented core domain entities and relationships using JPA  
- Built repository layer with `JpaRepository` for database operations  
- Enabled frontend integration with REST APIs and `@CrossOrigin` configuration  
- Implemented checkout workflow including order processing and tracking number generation  
- Developed service and controller layers for business logic  
- Added validation for customer input and improved data integrity  
- Created bootstrap data for testing and development  

## Key Features
- RESTful backend architecture  
- Relational data modeling with JPA  
- Checkout and order processing system  
- Angular frontend integration support
